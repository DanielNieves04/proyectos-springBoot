# Proyectos SpringBoot
Este repositorio contiene diferentes proyectos desarrollados con java / springboot / maven, diseñado para afianzar mis conocimientos.

Cada proyecto está estructurado para practicar:

- CRUD completo con buenas prácticas
- Validaciones y manejo de excepciones
- Arquitectura en capas (Controller → Service → Repository)
- Uso de JPA, relaciones, DTOs y más
- Test unitarios e integración en los proyectos más avanzados

# Objetivo
- Consolidar fundamentos de Spring Boot y Java Backend
- Manejar relaciones entre entidades, DTOs y validaciones complejas
